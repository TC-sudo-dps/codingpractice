### ⚒ 如何練習？
1. 點擊右上角的 **Fork**，複製一份到你的帳號。
2. 在你自己的倉庫中修改 `solution.py`。
3.請在終端機使用 python3 solution.py < tests/01.in > my_out && diff -Z my_out tests/01.out && echo "✅ 答案正確" || echo "❌ 答案錯誤" 來驗證答案
4. 如果看到 ✅，恭喜你通過了！
